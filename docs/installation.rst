Installation
============

Requirements
------------

* Python 3.7+
* Django 2.0+
* Bootstrap 3+


Install django-sorting-bootstrap
--------------------------------

You can install *Django Sorting Bootstrap* via pip_ from PyPI_:

.. code:: console

   $ pip install django-sorting-bootstrap


.. _PyPI: https://pypi.org/
.. _pip: https://pip.pypa.io/
